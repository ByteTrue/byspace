# Changelog

## v0.2.0-beta.1 - 2026-07-19

First clean-history BySpace beta, rebuilt from the Paseo `v0.2.0-beta.1` source snapshot (`0bec06c2db7d3ee071416cde80229eabd682b03e`).

- Retains the browser Web/PWA, CLI, local daemon, encrypted relay, and agent provider integrations.
- Removes Electron desktop, native iOS/Android, the marketing website, and Electron Browser automation.
- Renames the retained product, packages, CLI, configuration, state paths, and runtime environment to BySpace.
- Publishes `@bytetrue/byspace` and deploys the Web app and relay to ByteTrue Cloudflare infrastructure.
